# 🔐 Secure User Authentication System (Task-01)

## 📌 Internship Task
This project is submitted for **Task-01: Secure User Authentication System** under Prodigy Infotech Internship.

---

## 🚀 Project Overview
This module implements a secure user authentication system within a Grocery Store Management platform.

Users can register, log in securely, and access protected pages only after successful authentication. The system also supports different user roles such as Admin, Employee, and Customer.

---

## ✨ Features
- 🔑 User Registration (Signup)
- 🔐 Secure Login System
- 🧠 Session Management
- 👥 Role-Based Access Control (Admin / Employee / Customer)
- 🚫 Protected Routes (Unauthorized users cannot access restricted pages)
- ⚙️ Backend validation using PHP & MySQL

---

## 🛠 Tech Stack
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

---

## ⚙️ How to Run Locally

1. Install XAMPP and start:
   - Apache
   - MySQL

2. Copy the project folder into:
   ```
   C:/xampp/htdocs/
   ```

3. Import the database:
   - Open phpMyAdmin
   - Create a new database (e.g., `grocery`)
   - Import the `grocery.sql` file

4. Run the project:
   ```
   http://localhost/grocery store
   ```
